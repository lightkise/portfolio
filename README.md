## 项目简介

个人简历作品集网站，纯静态 HTML/CSS/JS 构建，无框架无后端，部署于 GitHub Pages。

## 功能特性

### ✨ 视觉特效
- 🎠 Hero 区 3D 旋转木马 · 鼠标悬停无需点击自动旋转
- 🎨 六种主题配色详情页 · 蓝 / 绿 / 棕 / 紫 / 钢铁蓝 / 青绿
- 🖱️ 技能标签悬停联动 · 对应作品卡片自动高亮浮现
- 🗺️ 跨境贸易全链路可视化流程图
- 💫 滚动触发淡入动画 · 数字跳动量化成果
- 🃏 卡片光泽扫过悬停效果
- 🔍 灯箱放大查看 · 长图支持上下滚动

### ⚙️ 技术亮点
- 纯原生 HTML / CSS / JavaScript · 零框架零依赖
- IntersectionObserver 实现滚动触发动画
- CSS perspective + rotateY 实现 3D 透视旋转
- mousemove 实时监听 · 鼠标位置映射旋转角度
- 惯性吸附算法 · 松手后自动对齐最近卡片
- CSS Grid 响应式布局 · 移动端触摸支持
- GitHub Pages 静态托管 · 零成本上线
